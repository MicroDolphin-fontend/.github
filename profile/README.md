## ▸ Overview

MicroDolphin is an AI-powered app builder that turns plain language into production-ready full-stack apps — deployed to real infrastructure with a shareable live URL in under 60 seconds.

Most AI builders give you a code dump. MicroDolphin gives you a **running product**.

- **▸** Multi-agent pipeline — intake, planning, generation, QA, deploy
- **▸** Real Kubernetes infrastructure — every project gets its own live URL
- **▸** RAG context memory — the AI remembers your project across sessions
- **▸** GitHub sync — one click, full commit history, no lock-in

## ▸ Repositories

### ◆ [`microdolphin`](https://github.com/firedolphin/microdolphin) — Core Engine `private`
Main generation pipeline. Prompt intake → multi-agent orchestration → code generation → QA → deploy.

**Stack:** Node.js · TypeScript · LLM pipeline · Kubernetes

### ◆ [`microdolphin-preview`](https://github.com/firedolphin/microdolphin-preview) — Live Infrastructure `private`
Provisions isolated containers and serves live preview URLs the moment code is ready.

**Stack:** Kubernetes · Docker · real-time hot-reload

### ◆ [`microdolphin-ui`](https://github.com/firedolphin/microdolphin-ui) — Frontend & Design System `private`
Product frontend plus the full brand and design system — tokens, components, Figma handoff.

**Stack:** Next.js · Archivo · Geist · glassmorphic design system

### ◆ `microdolphin-sdk` — TypeScript SDK `coming soon`
Public SDK for integrating MicroDolphin's generation API into your own tools.

### ◆ `microdolphin-docs` — Documentation `coming soon`
Public docs — API reference, guides, integration examples.

## ▸ How It Works

```
01  Describe    Plain language. Vague or specific — agents adapt either way.
02  Build       Seven agents run in parallel: intake, plan, write, QA, ship.
03  Preview     Live URL on real infra — before generation even finishes.
04  Iterate     Describe a change. Preview updates instantly. No redeploy.
05  Ship        Push to GitHub or share the live link. You're already done.
```

## ▸ Design System

The MicroDolphin design language is segmented, systematic, and glassmorphic. Everything is built from repeating units. No organic shapes. No arbitrary values.

```
Accent        #c4622d  — Ember. One per context. Never stack accents.
Background    #080808  — Void. Dark-first, always.
Secondary     #2d5fa0  — Ocean. Orbs and trust signals only.
Fonts         Archivo (display) · Geist Sans (body) · Geist Mono (data)
Radius        20px cards · 12px UI elements — never exceed 20px
Spacing       8pt grid — every value a strict multiple of 8
Glass         rgba(255,255,255, ≤0.08) · backdrop-filter blur(24px)
Border        rgba(255,255,255, 0.08) default · 0.18 hover
Gap           10px between all bento cells
```

## ▸ Security

- **▸** Every generated app runs in its own isolated container
- **▸** Code is never used for model training or shared with third parties
- **▸** JWT auth · SSO · SAML · RBAC available on Team plan
- **▸** ISO 27001-aligned practices across all infrastructure

**Found a vulnerability?** Email [security@microdolphin.com](mailto:security@microdolphin.com) — do not open a public issue. We acknowledge within 48 hours and provide a resolution timeline within 5 business days.
